# C++ Mastery Journey 🚀

A comprehensive repository containing my progress and learning journey through C++ programming. This repository covers everything from fundamental programming concepts to advanced Data Structures and Algorithms (DSA) and Object-Oriented Programming (OOP) patterns.

## 📖 Overview

This repository serves as a personal codebase and study guide for C++ concepts, specifically tailored towards technical interview preparation and building a strong foundation in software engineering principles.

## 🗂️ Project Structure & Topics

The repository contains numerous standalone C++ files, categorized by the core concepts they demonstrate:

### 1. C++ Fundamentals
- **Basics**: Variables, data types, namespaces, and standard I/O (`Basic concepts of c++.cpp`, `sample.cpp`, `Namespace Resolution.cpp`).
- **Mathematical Computations**: Calculating factorials (`Factorial.cpp`, `Factorial Function.cpp`), Fibonacci series (`Fibnacci Function.cpp`), powers (`Pow(x,n).cpp`), and prime numbers (`Prime no. calculator.cpp`).
- **Patterns**: Generating console structures like Butterfly (`Butterfly Pattern.cpp`), Floyd's Triangle (`Floyed's Triangle Pattern.cpp`), and Diamonds (`Hollow Diamond Pattern.cpp`).
- **Functions**: Function definitions, pass-by-value, pass-by-reference, and inline functions (`Pass by value.cpp`, `Pass by Reference .cpp`, `Inline Function.cpp`).

### 2. Memory Management & Pointers
- Demonstrates the difference between static and dynamic allocation using `new` and `delete` (`Static & Dynamic allocation.cpp`).
- Implementing pass-by-reference using pointers and memory addresses to modify variables actively (`Pass by Reference .cpp`).

### 3. Object-Oriented Programming (OOP)
- **Classes & Objects**: Defining private/protected data members and accessing them via public methods (`Classes Basic.cpp`, `Class with Data Member.cpp`).
- **Inheritance**: Implementations of Single, Multiple, Multilevel, Hierarchical, and Hybrid inheritance (`Single Inheritance basic.cpp`, `Multiple inheritance basic.cpp`, `Hybrid Inheritance basic.cpp`).
- **Polymorphism**: Compile-time (Function Overloading, Operator Overloading) and Runtime polymorphism (Function Overriding) (`Function Overloading.cpp`, `Operator Overloading.cpp`, `Function Overridding.cpp`).
- **Encapsulation**: Using scope resolution (`Scope Resolution operator.cpp`), private data shielding, and message passing (`Private Data & Message Passing.cpp`, `Object Interaction Concept.cpp`).
- **Mini-Projects**: 
  - Real-Time Student Application (`Real Time Student Application.cpp`, `Student Managment System.cpp`)
  - ATM Program (`Mini Real-World Program (ATM) .cpp`)

### 4. Standard Template Library (STL)
- Practical usage of `std::vector`, understanding standard library tools, and predefined functions (`Vectors Function Basics .cpp`, `Predefined vector function.cpp`).

### 5. Data Structures & Algorithms (DSA)
- **Arrays**: Sum of elements, reversing arrays, finding intersection, moving zeroes, and identifying duplicates (`Basics of Array.cpp`, `Moving Zeroes.cpp`, `Remove Duplicate Elements.cpp`).
- **Searching Algorithms**: Linear Search and Binary Search (`Linear Search Algorithm.cpp`, `Binary Search Algorithm.cpp`).
- **Sorting Algorithms**: Bubble Sort (`Bubble Sort.cpp`).
- **Optimized Algorithms**:
  - Kadane's Algorithm for Maximum Subarray Sum (`Kadane's Algorithm .cpp`)
  - Moore's Voting Algorithm for Majority Elements (`Majority element problem using Optimal approch.cpp`)
  - 2-Pointer methods for Pair Sum (`Pair Sum Problem by 2 pointer method.cpp`)
- **Advanced Data Structures**: Stack operations and Sparse Matrix handling (`Stack Operation.cpp`, `Sparse Matrix.cpp`).

## ⚙️ How to Compile and Run

Make sure you have a C++ compiler installed (like **GCC** using MinGW on Windows, or **Clang** on macOS). 

To run any individual file (e.g., `sample.cpp`):

```bash
g++ sample.cpp -o sample.exe
./sample.exe
```

3. **OOP Design**: Thoroughly review Inheritance patterns and Polymorphism.
