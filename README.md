## 🏃‍♂️ **Getting Started & Execution**

To test or run any code from this repository, ensure you have a C++ compiler (like `g++` on MinGW/Linux, or `Clang` on macOS) configured in your system path.

**1. Clone the project locally:**
```bash
git clone https://github.com/YourUsername/Cpp-Mastery.git
cd Cpp-Mastery
```

**2. Compile and execute any module:**
```bash
# Example compilation command using g++
g++ "Kadane's Algorithm .cpp" -o kadane_demo

# Run the compiled binary
# For Windows:
.\kadane_demo.exe
# For macOS/Linux:
./kadane_demo
```
